This theme is a child theme of Twenty Twenty-Four, and is set up to use FSE, or "Full Site Editing" with WordPress blocks. Because FSE has such a high level of customization, there's a solid chance that you would choose to use a child-theme such as this one rather than building a custom theme from scratch.

Start by reading the readme file to get up and running on your local environment. you will then likely want to take a look at the theme.json file and tailwind config file to start customizing some of the global styles.

This is the home page template, which can be edited under Appearance -> Editor -> Templates -> Front Page. The template file can also be edited at theme/template/front-page.html

The content above in the orange block is created from the Home Page edit area. This content is embedded within the template.
